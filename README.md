## Setup

```
#Download CopyCord
git clone https://github.com/hxntaish/copycord.git

#Navigate to project root 
cd copycord

#Install javascript packages
npm i

#Go to the bundle folder (bundle/src)
npm run setup

#Start the bundle server ( API, CDN, Gateway in one )
npm run start
```
